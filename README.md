## 📖 Описание

**Hockey Editor Pro** — десктопное приложение для покадрового анализа хоккейных матчей. Позволяет размечать видеозаписи игр, отмечая ключевые игровые события (голы, броски, вбрасывания, входы в зону и др.) с точностью до кадра.

Разработан для тренеров, аналитиков и скаутов, которым нужен быстрый и удобный инструмент для разбора игровых моментов.

### ✨ Основные возможности

#### 🎬 Работа с видео
- Поддержка форматов: **MP4, AVI, MOV, MKV, WMV, FLV**
- Покадровая навигация с помощью горячих клавиш
- Перемотка на ±5 секунд
- Регулировка скорости воспроизведения
- Drag & Drop — перетащите видео в окно

#### 🏷️ Система событий
- **13 предустановленных типов** хоккейных событий:

| Событие | Клавиша | Цвет |
|---------|:-------:|------|
| Гол | `G` | 🔴 |
| Бросок в створ | `H` | 🟠 |
| Бросок мимо | `M` | 🟡 |
| Заблокированный бросок | `B` | 🟤 |
| Вход в зону | `Z` | 🔵 |
| Выход из зоны | `X` | 🔵 |
| Вброс | `D` | 🔵 |
| Потеря | `T` | ⚪ |
| Перехват | `A` | 🟢 |
| Вбрасывание: Победа | `F` | 🟢 |
| Вбрасывание: Поражение | `L` | 🟢 |
| Блокшот в обороне | `K` | 🟣 |
| Удаление | `P` | 🟣 |

- **Пользовательские типы** — создавайте собственные категории событий
- Назначение горячих клавиш для каждого типа

#### ⏱️ Режимы записи
- **Фиксированная длина** — задайте длительность сегмента (по умолчанию 6 сек)
- **Динамический режим** — нажмите клавишу для начала, повторно для завершения
- Настраиваемые **Pre-roll** и **Post-roll** отступы

#### 📊 Аналитика
- **Статистика по типам** — количество, общее время, среднее время
- Визуальные полоски с цветовой кодировкой
- Фильтрация по типу события и наличию заметок

#### 🎯 Таймлайн
- Многодорожечный таймлайн с раздельными треками для каждого типа
- **Масштабирование** — от отдельных кадров до 2+ часов целиком
- Кнопка «⊞ Вместить всё» — уместить запись в ширину экрана
- Контекстное меню: редактировать, удалить, дублировать, экспортировать
- Цветная линейка времени (MM:SS / H:MM:SS)

#### ✏️ Редактор сегментов
- Покадровая подгонка точек входа/выхода (In/Out)
- Предпросмотр с зацикливанием
- Смена типа события, добавление заметок
- Навигация между сегментами ◀ ▶

#### 📤 Экспорт
- **Видео** — отдельные клипы или объединённый файл (H.264 / H.265)
- **CSV** — таблица сегментов для Excel / Google Sheets
- **PDF / HTML** — отчёт со статистикой
- Фильтрация сегментов перед экспортом
- Настраиваемое качество, разрешение, аудио

#### 💾 Проекты
- Сохранение / загрузка проектов (формат `.hep`)
- **Автосохранение** с настраиваемым интервалом
- Полная система **Undo / Redo** (Ctrl+Z / Ctrl+Y)
- Панель истории команд с навигацией кликом

#### 🎨 Интерфейс
- Тёмная тема
- Загрузочный экран (splash screen)
- Toast-уведомления
- Запоминание размеров и положения окна
- Полная **русская локализация**

### 📖 Как пользоваться (Quick Start)

#### 1. Открытие видео

Перетащите видеофайл в окно программы или выберите через меню **Файл → Открыть видео**.

#### 2. Разметка событий

Во время просмотра нажимайте горячие клавиши для создания маркеров:

```
Пример рабочего процесса:

  1. Запустите воспроизведение (Пробел)
  2. Увидели гол → нажмите G
  3. Увидели бросок → нажмите H
  4. Увидели удаление → нажмите P
  5. Маркеры автоматически создаются на таймлайне
```

> **Режим «Фиксированная длина»** (по умолчанию): маркер создаётся автоматически с длительностью 6 секунд от момента нажатия.
>
> **Режим «Динамический»**: первое нажатие — начало записи, второе — конец.

#### 3. Редактирование

- Двойной клик по сегменту на таймлайне → откроется редактор
- Покадровая подгонка точек In/Out
- Смена типа события, добавление заметки

#### 4. Экспорт

Меню **Экспорт** (Ctrl+E):
- **Видео** — нарезка выбранных сегментов в отдельные клипы или один файл
- **CSV** — таблица для Excel
- **PDF** — отчёт со статистикой для тренерского штаба

#### 5. Сохранение

`Ctrl+S` — сохранить проект. Автосохранение работает каждую минуту.

### ❓ FAQ

<details>
<summary><b>Какие форматы видео поддерживаются?</b></summary>

MP4, AVI, MOV, MKV, WMV, FLV — любые форматы, которые поддерживает OpenCV. Рекомендуется MP4 (H.264) для лучшей производительности.
</details>

<details>
<summary><b>Нужен ли FFmpeg?</b></summary>

FFmpeg нужен **только для экспорта видео**. Просмотр, разметка и сохранение проектов работают без него.

Скачать: [ffmpeg.org/download.html](https://ffmpeg.org/download.html)

После установки убедитесь, что `ffmpeg` доступен в командной строке:
```bash
ffmpeg -version
```
</details>

<details>
<summary><b>Какой максимальной длины может быть видео?</b></summary>

Ограничений нет. Таймлайн поддерживает масштабирование от отдельных кадров до 3+ часов. Кнопка **«⊞ Вместить всё»** уместит любую длительность в ширину экрана.
</details>

<details>
<summary><b>Можно ли использовать для других видов спорта?</b></summary>

Да. Хотя предустановленные типы событий рассчитаны на хоккей, вы можете создать **собственные типы** через Настройки → Типы событий. Подойдёт для футбола, баскетбола, волейбола и др.
</details>

<details>
<summary><b>Где хранятся настройки и проекты?</b></summary>

- Настройки: `%LOCALAPPDATA%/HockeyEditor/settings.json`
- Проекты: в файлах `.hep` в выбранной вами директории
- Автосохранение: рядом с файлом проекта
</details>

<details>
<summary><b>Можно ли работать без установки (portable)?</b></summary>

Да. После сборки через `build.bat` папку `dist/HockeyEditor/` можно скопировать на флешку и запускать `HockeyEditor.exe` на любом компьютере с Windows 10/11.
</details>

<details>
<summary><b>Поддерживается ли macOS или Linux?</b></summary>

Приложение написано на PySide6 (Qt) и теоретически кроссплатформенное. Однако тестирование проводилось только на Windows. Сборка EXE и установщик — только для Windows.
</details>

### 🔧 Решение проблем

<details>
<summary><b>❌ Видео не открывается</b></summary>

**Причина:** Кодек видео не поддерживается OpenCV.

**Решение:**
1. Переконвертируйте видео в MP4 (H.264):
   ```bash
   ffmpeg -i input.avi -c:v libx264 -crf 23 output.mp4
   ```
2. Убедитесь, что `opencv-python` установлен:
   ```bash
   pip install opencv-python --upgrade
   ```
</details>

<details>
<summary><b>❌ Экспорт видео завершается с ошибкой</b></summary>

**Причина:** FFmpeg не установлен или не в PATH.

**Решение:**
1. Скачайте FFmpeg: [ffmpeg.org/download.html](https://ffmpeg.org/download.html)
2. Добавьте путь к `ffmpeg.exe` в системную переменную PATH
3. Перезапустите приложение
4. Проверьте:
   ```bash
   ffmpeg -version
   ```
</details>

<details>
<summary><b>❌ Приложение зависает при открытии большого видео</b></summary>

**Причина:** Недостаточно оперативной памяти.

**Решение:**
- Закройте другие приложения
- Используйте видео с меньшим разрешением (720p вместо 4K)
- Переконвертируйте:
  ```bash
  ffmpeg -i input_4k.mp4 -vf scale=-2:720 -c:v libx264 -crf 23 output_720p.mp4
  ```
</details>

<details>
<summary><b>❌ Не создаётся PDF отчёт</b></summary>

**Причина:** Библиотека `reportlab` не установлена.

**Решение:**
```bash
pip install reportlab
```
Если не установлена — отчёт автоматически создаётся в формате HTML (открывается в браузере).
</details>

<details>
<summary><b>❌ Иероглифы в CSV файле (Excel)</b></summary>

**Причина:** Excel не распознаёт кодировку UTF-8.

**Решение:** CSV экспортируется в UTF-8 с BOM — Excel 2016+ открывает корректно. Для старых версий:
1. Откройте Excel → Данные → Из текста/CSV
2. Выберите кодировку UTF-8
3. Разделитель: точка с запятой (`;`)
</details>

<details>
<summary><b>❌ Ошибка при сборке EXE</b></summary>

**Решение:**
```bash
# Обновить PyInstaller
pip install pyinstaller --upgrade

# Пересобрать с очисткой
pyinstaller hockey_editor.spec --clean --noconfirm
```

Если ошибка связана с `cv2`:
```bash
pip install opencv-python-headless
```
</details>
