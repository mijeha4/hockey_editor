# Реализация функциональности удаления сегментов

## Обзор

Реализована полноценная система удаления сегментов в Hockey Editor с поддержкой:
- Удаления по нажатию клавиши Delete
- Выделения сегментов в таймлайне и списке сегментов
- Отмены и повтора операций удаления
- Интеграции с существующей MVC архитектурой

## Реализованные компоненты

### 1. Команда удаления маркера (`DeleteMarkerCommand`)

**Файл:** `src/controllers/timeline_controller.py`

**Описание:** Команда для undo/redo системы, реализующая паттерн Command.

**Ключевые особенности:**
- Сохраняет состояние маркера перед удалением
- Поддерживает отмену и повтор операции
- Интегрирована с системой истории команд
- Вызывает сигналы обновления интерфейса

**Методы:**
- `execute()`: Удаляет маркер и сохраняет его для отмены
- `undo()`: Восстанавливает удаленный маркер
- `redo()`: Повторяет удаление маркера

### 2. Обработка Delete shortcut в MainWindow

**Файл:** `src/views/windows/main_window.py`

**Описание:** Обработка нажатия клавиши Delete в главном окне.

**Логика работы:**
1. Проверка выделения в таблице сегментов
2. Если выделен сегмент в таблице - удаление по его индексу
3. Если ничего не выделено в таблице - удаление последнего добавленного сегмента
4. Поддержка Undo/Redo через соответствующие сигналы

**Методы:**
- `_on_shortcut_pressed()`: Обработка всех горячих клавиш
- `_delete_selected_segment()`: Логика удаления выделенного сегмента
- `_delete_marker_by_index()`: Вызов удаления через контроллер

### 3. Выделение сегментов в TimelineWidget

**Файл:** `src/views/widgets/timeline.py`

**Описание:** Реализация выделения сегментов при клике на таймлайне.

**Изменения:**
- Добавлен сигнал `segment_selected` в `TimelineGraphicsScene`
- Модифицирован `mousePressEvent` для обработки кликов на сегментах
- Реализована визуальная подсветка выделенных сегментов
- Автоматическое снятие выделения с других сегментов

**Методы:**
- `mousePressEvent()`: Обработка кликов для выделения сегментов
- `SegmentGraphicsItem.paint()`: Визуальное отображение выделенного состояния

### 4. Выделение сегментов в SegmentListWidget

**Файл:** `src/views/widgets/segment_list.py`

**Описание:** Реализация выделения строк в таблице сегментов.

**Изменения:**
- Добавлен обработчик `_on_item_clicked()` для выделения строк
- Интеграция с системой выделения Qt таблиц

**Методы:**
- `_on_item_clicked()`: Выделение строки при клике

## Архитектурные особенности

### MVC Интеграция

Система полностью интегрирована в существующую MVC архитектуру:

1. **Model**: `Marker` класс хранит данные сегмента
2. **View**: `TimelineWidget` и `SegmentListWidget` отображают сегменты
3. **Controller**: `TimelineController` управляет операциями удаления

### Сигналы и слоты

Используется система сигналов Qt для связи компонентов:

```python
# Сигналы в TimelineController
marker_deleted = Signal(int)  # Индекс удаленного маркера
markers_changed = Signal()    # Изменение списка маркеров

# Сигналы в MainWindow
delete_requested = Signal(int)  # Запрос на удаление маркера
undo_requested = Signal()       # Запрос на отмену
redo_requested = Signal()       # Запрос на повтор
```

### Undo/Redo система

Реализована через паттерн Command с интеграцией в существующую систему истории:

```python
# В TimelineController
from services.history.history_manager import get_history_manager

def delete_marker(self, marker_idx: int):
    """Delete marker with undo/redo support."""
    command = DeleteMarkerCommand(self, marker_idx)
    self.history_manager.execute(command)
```

## Работа с выделением

### Приоритет выделения

1. **Таблица сегментов**: Если выделена строка в таблице - удаляется этот сегмент
2. **Таймлайн**: Если выделен сегмент на таймлайне - удаляется этот сегмент
3. **Fallback**: Если ничего не выделено - удаляется последний добавленный сегмент

### Визуальное выделение

- **Таймлайн**: Белая рамка вокруг выделенного сегмента
- **Таблица**: Стандартное выделение строки в Qt таблице

## Использование

### Для пользователя

1. **Выделение сегмента**: Клик на сегмент в таймлайне или таблице
2. **Удаление**: Нажатие клавиши Delete
3. **Отмена**: Ctrl+Z
4. **Повтор**: Ctrl+Y

### Для разработчика

```python
# Создание и выполнение команды удаления
from controllers.timeline_controller import DeleteMarkerCommand

controller = TimelineController(main_controller)
command = DeleteMarkerCommand(controller, marker_index)
controller.history_manager.execute(command)

# Отмена операции
controller.history_manager.undo()

# Повтор операции
controller.history_manager.redo()
```
