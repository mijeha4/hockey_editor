# Руководство: Фильтрация по выбранному маркеру

## Описание функции

Новая функция позволяет временно показывать только один выбранный сегмент на таймлайне, скрывая все остальные. Это удобно для фокусировки на конкретном моменте матча.

## Как использовать

1. **Выбор сегмента**: Кликните левой кнопкой мыши по любому сегменту на таймлайне
2. **Показ только выбранного**: После клика останется виден только выбранный сегмент
3. **Возврат к просмотру всех**: Кликните по тому же сегменту еще раз, чтобы показать все сегменты
4. **Выбор другого сегмента**: Кликните по другому сегменту, чтобы показать только его

## Поведение

- **Режим "all"**: Показываются все сегменты (режим по умолчанию)
- **Режим "selected"**: Показывается только один выбранный сегмент
- **Переключение**: Клик по сегменту переключает между режимами
- **Смена выбора**: Клик по другому сегменту меняет выбранный сегмент

## Техническая реализация

### Измененные файлы

1. **`src/controllers/timeline_controller.py`**
   - Добавлены поля `filter_mode` и `selected_marker_id` для управления фильтрацией
   - Реализован метод `toggle_selected_mode()` для переключения режимов
   - Реализован метод `get_filtered_markers()` для получения отфильтрованных маркеров
   - Метод `_update_markers_display()` обновляет отображение на таймлайне и в списке сегментов

2. **`src/views/widgets/timeline.py`**
   - Модифицирован метод `mousePressEvent()` для вызова `toggle_selected_mode()` при клике на сегмент

### Преимущества упрощенного подхода

- **Минимум кода**: Не требуется отдельный FilterController
- **Простая логика**: Только два режима - "all" и "selected"
- **Быстрая реакция**: Нет сложной синхронизации между компонентами
- **Легко поддерживать**: Меньше точек отказа и зависимостей

## Пример использования

```python
# Создаем контроллер
controller = TimelineController(project, timeline_widget, segment_list_widget, ...)

# Изначально показываются все маркеры
filtered = controller.get_filtered_markers()  # Вернет все 3 маркера

# Выбираем первый маркер
controller.toggle_selected_mode(0)  # Режим: selected, ID: 1

# Теперь виден только первый маркер
filtered = controller.get_filtered_markers()  # Вернет только 1 маркер с ID=1

# Выбираем второй маркер
controller.toggle_selected_mode(1)  # Режим: selected, ID: 2

# Теперь виден только второй маркер
filtered = controller.get_filtered_markers()  # Вернет только 1 маркер с ID=2

# Кликаем по второму маркеру еще раз - возвращаемся к просмотру всех
controller.toggle_selected_mode(1)  # Режим: all, ID: None

# Снова видны все маркеры
filtered = controller.get_filtered_markers()  # Вернет все 3 маркера
```
