# Сборка Hockey Editor Pro в исполняемый файл (.exe)

## Что было сделано

✅ **Успешно создан исполняемый файл** `Hockey_Editor_Pro.exe` (8.3 МБ)
✅ Настроена автоматическая сборка с PyInstaller
✅ Включены все основные зависимости (PySide6, OpenCV, NumPy, etc.)

## Файлы для сборки

- `requirements.txt` - обновлен с PyInstaller
- `setup.py` - скрипт установки для проекта
- `hockey_editor.spec` - спецификация для PyInstaller
- `dist/Hockey_Editor_Pro.exe` - готовый исполняемый файл

## Как пересобрать exe

```bash
# Установить зависимости
pip install -r requirements.txt

# Собрать exe
pyinstaller --clean hockey_editor.spec
```

## Результат

- **Размер**: ~8.3 МБ (включает все зависимости)
- **Совместимость**: Windows 10/11 (64-bit)
- **Зависимости**: Полностью standalone, не требует установки Python
